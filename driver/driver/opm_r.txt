                      Руководство оператора
1 Назначение программы 
Программа driver.exe предназначена для создания и работы с объектами семи классов: базовый 
“Человек” (Person) и производный от него “Водитель” (Driver), включенный в него абстрактный класс 
«Водительские права» (License Tag), от которого наследуются два класса: «Б1» (B1) и «БЕ» (BE),
 производный от них класс «Б» (B), дружественный класс «Машина» (Auto). 
Класс Person имеет поля FirstName (название), LastName (фамилия), Otchestvo (отчество),
 Age (возраст). Для класса Person предусмотрены действия: создание объектов, вывод на 
системное устройство вывода содержимого всех данных-членов класса созданных объектов, 
перегрузка бинарного оператора >(больше), который сравнивает объект-левого операнда значение 
вoзраста со значением объекта-правого операнда и выводит соответствующее сообщение, и также 
выводит сообщение о том, можно ли получить водительские права или нет. 
Класс Driver имеет дополнительные поля Code (код категории) и Stage (стаж). Для класса Driver
 предусмотрены действия: создание объектов, вывод на системное устройство вывода содержимого
 всех данных-членов класса этого объекта, перегрузка унарного оператора ++, который увеличивает 
стаж вождения водителя и его возраст на один год. 
Также программа выводит на системное устройство вывода по запросу пользователя условие задачи и 
содержимое этого документа. 
Г.2 Условия выполнения программы 
Для выполнения программы необходима ПЭВМ типа IBM PC или программно совместимая с ней,
 функционирующая под управлением MS-DOS (Windows 9x/200x). 
Г.3 Выполнение программы 
Исполнимый файл driver.exe и все файлы данных должны быть размещены в одном каталоге. Запуск 
программы выполняется с помощью вызова исполнимого файла driver.exe (в среде MS-DOS или Windows 9x/200x). 
После запуска программы на экране появляется окно заставки, которое содержит общую информацию
 о теме курсовой работы и авторе. В нижней части окна заставки размещены два сообщения: 
Нажмите m для выхода в главное меню и l - для смены языка. 
При нажатии на l происходит смена языка с английского (по умолчанию) на русский. При нажатии на m
 в окне отображается главное меню: 
1. Описание заданной области задачи.2. Работа с объектами.3. Помощь.4. Выход. 
Выбор опции главного меню осуществляется с помощью ввода номера соответствующей опции (1, 2, 3, 4). 
При выборе в главном меню опции “Описание заданной области задачи” (№1) на экран выводится содержимое 
файла condition_a.txt (condition_r.txt) со словесной постановкой задачи курсовой работы. При выборе 
в главном меню опции “ Работа с объектами ” (№2) на экран выводится меню работы с объектами: 1.Пользователь.
2. Водитель.3. Записать объекты в файл.4. Прочитать все данные из файла.5. Закончить работу с объектами. 
Если пользователь нажмет цифру несуществующего пункта меню, он получит сообщение об ошибке.  
При выборе опции “Пользователь ” (№1): Появится меню Пользователя: 1.Создать объекты. 
2. Показать объекты. 3. Записать объекты в файл.4. Прочитать все данные из файла. 5. Закончить работу с объектами. 
При выборе пункта «Создать объекты» (№1)  появится сообщение: «У вас есть водительские права?(у for  yes): ». 
Если будет нажата кнопка у, начнется создание объектов классов Driver, Водительские права и Машины.
 В противном случае, будет создан объект класса Person. 
При выборе опции “Показать объекты” (№2) на экран выводятся с помощью метода print значения всех
 данных-членов всех созданных классов. При выборе опции «Записать данные в файл» (№3) с помощью 
перегруженных операторов вывода будет осуществлена запись всех объектов системы в файл. При выборе
 опции «Прочитать все данные из файла» (№4) будет выведено на экран содержимое файла. При выборе пункта
 «Закончить работу с объектами» (№5) или при нажатии кнопки esc будет осуществлен возврат в меню. 
При выборе пункта «Водитель» (№2) в окне появится меню объектов: 
1. Работа с объектами. 2. Показать сведения объектов.  3. Выход.  
При выборе опции “Работа с объектами” (№1) будет предложено несколько действий: 
1. Увеличить стаж вождения водителя(оператор ++), 2. Сравнить возрасты людей. 3. Получить средний возраст людей. 
4. Ф.И.О. человека. 
При выборе пункта «Увеличить стаж вождения» (№1) выполняется оператор ++,
 и на экран выводится сообщение о выборе водителя, у которому нужно увеличить стаж. 
Автоматически с увеличением стажа, увеличится и возраст водителя. 
При выборе пункта «Сравнить возрасты человек» (№2) у пользователя запросят порядковые номера 2-х
 людей, и с помощью оператора > произойдет сравнение заданных людей,  отображением результата в виде 
сообщений. При выборе опции “Получить средний возраст ” (№3) на экран выведется средний возраст всех
созданных объектов класса Person. При выборе опции “ Ф.И.О. человека” (№4) при выборе человека будет 
выведена фамилия, имя и отчество данного человека. При выборе опции «Выход»(№5) выполняется
возврат в меню объектов. При выборе пункта «Показать сведения об объектах»(№2) будут показаны
все данные-члены, существующие в данный момент в системе.
При выборе в главном меню опции “Помощь” (№3) на экран выводится содержимое файла help_h_r.txt (help_h_a.txt). 
Это меню содержит такие пункты: 
1. Помощь. 2. О программе (Описание). 3. Об авторе. 4. Руководство оператора.  5. Условие.
При выборе соответствующих пунктов меню помощи будет отображаться текст, содержащийся в файлах.
При выборе в главном меню опции “Выход” (№4) завершается работа программы. 
При задании неверного номера опции главного меню предлагается повторить выбор опции.
